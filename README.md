# Decision Support System for Healthy Diet Recommendation Using SAW

## 📖 Overview

This project is a web-based Decision Support System (DSS) developed as a final-year project to provide healthy diet recommendations using the Simple Additive Weighting (SAW) method.

The application helps users obtain diet recommendations based on predefined criteria, while administrators can manage criteria, alternatives, recommendation results, and user data through an admin dashboard.

---

## ✨ Features

- User-friendly interface
- Healthy diet recommendation
- Simple Additive Weighting (SAW) calculation
- Criteria management
- Alternative management
- Recommendation history
- Admin dashboard
- User management

1. Clone this repository.

```bash

```

2. Move the project folder to **htdocs**.

3. Import the database from the SQL file.

4. Configure the database connection.

5. Start Apache and MySQL.

6. Open the project in your browser.

```
http://localhost/project-folder
```

## 📚 Method

The system uses the **Simple Additive Weighting (SAW)** method to calculate and rank healthy diet recommendations based on predefined criteria and weights.
